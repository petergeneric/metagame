Getting Started

Welcome to the NY Noir Maps interactive directory. This guide will help you navigate the interface and find information quickly.

The Sidebar

The sidebar on the left contains five search modes, each accessible via the tab buttons at the top:

White Pages — Search for people by name. Type the beginning of a surname to find matching entries. Results are sorted alphabetically.
Yellow Pages — Browse the business directory by category. Select a category from the dropdown to see all matching businesses. Use "See Also" links to explore related categories.
Reverse Directory — Look up locations by block. Enter a building code like LI-20 to see all listings at that address. You can also click on a block on the map to be shown this listing.
Advanced — Use the full query language to perform complex searches with filters for location, distance, and properties. See the Advanced Search help topic for syntax details.
Bookmarks — View and manage your saved locations. Add bookmarks by clicking the bookmark icon in any search result. Bookmarks can include notes and labels.

The Map

The map displays search results as pins. Click a pin to see details about that location. Use your mouse or touch gestures to pan and zoom.

  • Green pins show businesses
  • Red, Brown, and Purple dots indicate people. Purple for people in apartment buildings, Brown for people in a duplex, Red for all others.
  • Large retangular markers denote a bookmark (or author mode anchor pin)

Quick Tips

  • Click any lead number (e.g., 1-1234) in results to highlight it on the map, and to display the Action Bar
  • Use the search box in this help window to find specific topics
  • Press at any time to open this help dialog
